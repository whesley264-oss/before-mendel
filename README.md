# 🧬 Proto-Genetics — Experiência Interativa

Este projeto não é um site comum.

É uma experiência digital interativa que explora como a hereditariedade era compreendida antes das descobertas de Gregor Mendel.

A proposta é transformar um conteúdo tradicionalmente estático em uma narrativa visual imersiva, onde o usuário não apenas lê, mas percorre a evolução das ideias ao longo do tempo.

## 🎯 Objetivo

Apresentar conceitos de hereditariedade pré-Mendel de forma clara, envolvente e visual, destacando:

- Teorias antigas e filosóficas  
- Erros e limitações do conhecimento da época  
- A transição para o pensamento científico  
- O surgimento das ideias que prepararam o caminho para Mendel  

## 🎬 Abordagem

O projeto foi desenvolvido como uma experiência baseada em scroll, inspirada em documentários interativos.

Cada seção funciona como um "capítulo", com transições suaves, animações e progressão narrativa, criando a sensação de continuidade e descoberta.

## ⚙️ Estrutura

- Página única (One Page)
- Navegação por scroll progressivo  
- Animações orientadas por interação  
- Design focado em contraste e legibilidade  

## 🧠 Conceito

A ideia central é simples:

> A ciência não nasce pronta. Ela evolui a partir de erros, tentativas e interpretações limitadas.

Este projeto busca mostrar exatamente isso.

## 🚀 Diferencial

Enquanto apresentações tradicionais focam apenas no conteúdo, este projeto prioriza a experiência do usuário, tornando o aprendizado mais intuitivo e memorável.
