# UX FLOW — EXPERIÊNCIA DO USUÁRIO

## 🎯 OBJETIVO

Guiar o usuário através de uma jornada clara e envolvente.

---

## 🔄 FLUXO

1. Curiosidade (início impactante)
2. Descoberta (conteúdo revelado)
3. Entendimento (explicações claras)
4. Conclusão (reflexão final)

---

## 📜 SCROLL

- Deve ser suave
- Não deve travar
- Deve guiar naturalmente para a próxima seção

---

## 🧠 SENSAÇÃO

O usuário deve sentir que está explorando algo, não apenas lendo.

---

## 🚫 PROIBIDO

- Confusão na navegação
- Quebra de fluxo
- Excesso de informação de uma vez
